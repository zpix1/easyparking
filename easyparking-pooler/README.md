# Pooler

## Commands
### Install Dependencies
```
mix deps.get
```
### Run Server For Development
```
iex -S mix phx.server
```
### Format
```
mix format
```
### Run Tests
```
mix test
```
### Test Coverage
```
mix coveralls
```
### Static Code Analysis - Consistency
```
mix credo --strict
```
### Static Code Analysis - Types
```
mix dialyzer
```
### Docs Quality Check
```
mix inch
```
### Docs Generation
```
mix docs
```
